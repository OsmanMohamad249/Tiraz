About Qeyafa Application

ما هو طراز؟ / What is Qeyafa?

طراز هو تطبيق ويب نموذجي متكامل مبني باستخدام Flask وPython. يوفر التطبيق نظاماً لإدارة العناصر مع إمكانية تصنيفها حسب الأنماط.

Qeyafa is a complete model web application built with Flask and Python. The application provides a system for managing items with the ability to categorize them by styles.

التقنيات المستخدمة / Technologies Used

  • Flask - Python web framework
  • SQLAlchemy - Database ORM
  • HTML/CSS/JavaScript
  • SQLite Database

الهيكل المعماري / Architecture

يتبع التطبيق نمط MVC (Model-View-Controller) لتنظيم الكود وسهولة الصيانة.

The application follows the MVC (Model-View-Controller) pattern for code organization and maintainability.
